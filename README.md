# 대전전단지 제작 웹사이트

## 프로젝트 개요
대전 지역의 전단지, 현수막, 명함 제작 및 배포 서비스를 제공하는 하이엔드 원페이지 웹사이트입니다.

## 주요 특징

### SEO 최적화
- ✅ 메타 태그 최적화 (title, description, keywords)
- ✅ Open Graph 및 Twitter 카드 설정
- ✅ Canonical URL 설정
- ✅ 시맨틱 HTML5 마크업
- ✅ 적절한 H1, H2, H3 태그 구조
- ✅ 모든 이미지에 alt 속성 포함

### GEO (지역 최적화)
- ✅ 대전 전지역 키워드 포함 (유성구, 서구, 중구, 동구, 대덕구)
- ✅ 둔산동 학원가, 은행동 상권 등 구체적 지역명 포함
- ✅ 대전 소상공인 맞춤 서비스 강조

### AEO (답변 엔진 최적화)
- ✅ LocalBusiness Schema.org 구조화 데이터
- ✅ FAQPage Schema.org 구조화 데이터
- ✅ 상세한 FAQ 섹션

### 기술 스택
- HTML5 (시맨틱 마크업)
- Tailwind CSS (CDN 버전)
- Vanilla JavaScript
- Formspree (폼 처리)

### 핵심 기능
1. **무료 디자인 상담 양식** (Formspree 연동)
   - 업체명, 담당자명, 연락처, 서비스 유형, 문의내용
   - JavaScript 유효성 검사
   - 실시간 전화번호 포맷팅

2. **간편 전화문의 CTA**
   - 고정 헤더에 전화번호 버튼
   - 모바일 `tel:` 링크 지원
   - 전화번호: 010-2364-2355

3. **업종별 포트폴리오**
   - 탭 기반 갤러리 (식당/카페, 학원/교육, 병원/뷰티, 부동산/기타)
   - 반응형 이미지 그리드

4. **반응형 디자인**
   - 모바일 최적화
   - 태블릿 및 데스크톱 지원
